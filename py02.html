조건문반복문흐름 제어문제 해결

🎯 학습목표

영역목표
지식 목표if, elif, else, for, while, break, continue, range()의 의미를 설명할 수 있다.
기능 목표조건과 반복을 이용하여 홀짝 판별, 구구단, 팩토리얼, 별 찍기, 숫자 맞추기 프로그램을 작성할 수 있다.
태도 목표반복되는 일을 자동화하고 조건에 따라 다르게 처리하는 컴퓨팅 사고를 기른다.

🧠 핵심 개념

1. 조건문

조건문은 어떤 조건이 참인지 거짓인지에 따라 실행할 코드를 선택하는 구조입니다. 예: 점수가 60점 이상이면 합격, 아니면 불합격.

2. 반복문

반복문은 같은 작업을 여러 번 자동으로 실행하는 구조입니다. for문은 정해진 횟수 반복에, while문은 조건이 참인 동안 반복에 자주 사용됩니다.

3. 흐름 제어

break는 반복을 즉시 종료하고, continue는 이번 반복의 남은 부분을 건너뛰고 다음 반복으로 넘어갑니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. if 기본 구조

score = 85
if score >= 60:
    print("합격")

score가 60 이상이면 조건이 참이므로 "합격"이 출력됩니다.

코드 2. if else 구조

age = 15
if age >= 18:
    print("성인")
else:
    print("청소년")

조건이 참이면 if 블록, 거짓이면 else 블록이 실행됩니다.

코드 3. elif 여러 조건

score = 92
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")

여러 조건을 위에서부터 검사합니다. 처음 참이 되는 조건의 코드만 실행됩니다.

코드 4. 중첩 조건문

score = 95
attend = True
if score >= 90:
    if attend:
        print("우수상")

조건문 안에 조건문을 넣어 더 세밀한 판단을 할 수 있습니다.

코드 5. for문과 range()

for i in range(1, 6):
    print(i)

range(1, 6)은 1부터 5까지의 숫자를 만듭니다. 끝값 6은 포함되지 않습니다.

코드 6. 구구단

dan = 3
for i in range(1, 10):
    print(dan, "x", i, "=", dan*i)

반복문을 이용하면 3단의 1부터 9까지 곱셈 결과를 자동 출력할 수 있습니다.

코드 7. while문

n = 1
while n <= 5:
    print(n)
    n += 1

조건 n <= 5가 참인 동안 반복합니다. n을 증가시키지 않으면 무한 반복될 수 있습니다.

코드 8. break

for i in range(1, 10):
    if i == 5:
        break
    print(i)

i가 5가 되면 break로 반복문이 종료됩니다. 따라서 1,2,3,4만 출력됩니다.

코드 9. continue

for i in range(1, 6):
    if i == 3:
        continue
    print(i)

i가 3일 때 print를 건너뛰므로 1,2,4,5가 출력됩니다.

코드 10. 팩토리얼

result = 1
for i in range(1, 6):
    result *= i
print(result)

1×2×3×4×5를 계산하여 5! = 120을 출력합니다.

🌟 수업마무리하기

오늘 배운 조건문과 반복문은 프로그래밍에서 가장 중요한 흐름 제어 도구입니다.

  • 조건문은 선택을 만든다.
  • 반복문은 자동화를 만든다.
  • break와 continue는 반복의 흐름을 세밀하게 조절한다.
  • 조건과 반복을 결합하면 게임, 계산기, 데이터 처리 프로그램을 만들 수 있다.

핵심 정리: 프로그램은 입력된 데이터를 조건에 따라 판단하고, 필요한 작업을 반복하면서 문제를 해결한다.
