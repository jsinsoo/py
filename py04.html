🧠 핵심 개념

1. 함수

함수는 특정 작업을 수행하는 코드 묶음입니다. 함수에 이름을 붙이면 필요할 때마다 다시 호출할 수 있습니다.

2. 매개변수와 인수

매개변수는 함수가 값을 받을 자리이고, 인수는 함수를 호출할 때 실제로 전달하는 값입니다.

3. 반환값

return은 함수의 처리 결과를 함수 밖으로 돌려보냅니다. 반환값이 있으면 다른 계산에 다시 활용할 수 있습니다.

4. 지역변수와 전역변수

함수 안에서 만든 변수는 보통 지역변수이고, 함수 밖에서 만든 변수는 전역변수입니다.

5. 모듈

모듈은 함수와 변수 등을 모아 놓은 파일입니다. import를 사용해 다른 사람이 만든 기능도 가져와 쓸 수 있습니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. 함수 정의와 호출

def hello():
    print("Hello Python")

hello()

def로 함수를 정의하고, hello()처럼 함수 이름 뒤에 괄호를 붙여 호출합니다.

코드 2. 매개변수

def greet(name):
    print("안녕하세요", name)

greet("민수")

name은 매개변수입니다. "민수"라는 인수가 name 자리에 들어갑니다.

코드 3. 반환값

def add(a, b):
    return a + b

print(add(3, 5))

return은 계산 결과를 함수 밖으로 돌려줍니다. add(3,5)는 8이 됩니다.

코드 4. 기본값 매개변수

def power(x, n=2):
    return x ** n

print(power(3))
print(power(3, 3))

n=2처럼 기본값을 정하면 두 번째 값을 생략했을 때 자동으로 2가 사용됩니다.

코드 5. 지역변수

def test():
    x = 10
    print(x)

test()

x는 함수 안에서 만들어진 지역변수입니다. 함수 안에서만 안전하게 사용됩니다.

코드 6. 전역변수 참조

x = 100

def show():
    print(x)

show()

함수 밖에서 만든 x를 함수 안에서 읽을 수 있습니다. 이런 변수는 전역변수입니다.

코드 7. lambda

square = lambda x: x * x
print(square(5))

lambda는 짧은 익명 함수를 만들 때 사용합니다. 여기서는 제곱 함수를 만듭니다.

코드 8. map

nums = [1, 2, 3]
result = list(map(lambda x: x * 2, nums))
print(result)

map은 리스트의 각 원소에 같은 함수를 적용합니다. 각 숫자가 2배가 됩니다.

코드 9. filter

nums = [1, 2, 3, 4, 5]
even = list(filter(lambda x: x % 2 == 0, nums))
print(even)

filter는 조건을 만족하는 원소만 남깁니다. 짝수만 추출됩니다.

코드 10. zip과 import

import math

names = ["A", "B", "C"]
scores = [90, 80, 70]

print(list(zip(names, scores)))
print(math.sqrt(16))

zip은 여러 자료를 짝지어 줍니다. import math로 수학 모듈을 가져와 sqrt()를 사용할 수 있습니다.

🌟 수업마무리하기

함수와 모듈은 코드를 짧게 만드는 기술이 아니라, 생각을 구조화하고 재사용 가능한 프로그램을 만드는 핵심 도구입니다.

  • def는 함수를 만든다.
  • 매개변수는 함수가 외부 값을 받는 통로이다.
  • return은 결과를 돌려준다.
  • lambda, map, filter, zip은 데이터를 간결하게 처리하는 도구이다.
  • import는 외부 모듈의 기능을 가져온다.

핵심 정리: 좋은 함수는 한 가지 일을 명확하게 수행하고, 필요할 때 다시 사용할 수 있어야 합니다.
