🧠 핵심 개념

1. NumPy

NumPy는 수치 계산을 빠르게 처리하는 배열 기반 라이브러리입니다. 평균, 합계, 벡터 연산 등에 활용됩니다.

2. Pandas

Pandas는 표 형태 데이터를 다루는 라이브러리입니다. DataFrame을 사용해 행과 열 구조의 데이터를 쉽게 분석합니다.

3. 데이터 전처리

전처리는 결측값, 중복, 공백, 이상값 등을 정리하여 분석 가능한 데이터로 만드는 과정입니다.

4. 시각화

Matplotlib과 Seaborn은 데이터를 그래프로 표현하여 패턴과 차이를 쉽게 이해하도록 돕습니다.

5. 머신러닝

머신러닝은 데이터에서 규칙을 학습하여 새로운 데이터의 결과를 예측하는 기술입니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. NumPy 배열과 평균

import numpy as np

scores = np.array([80, 90, 100])
print(scores.mean())

NumPy 배열은 숫자 데이터를 빠르게 계산합니다. 평균은 90.0입니다.

코드 2. Pandas DataFrame

import pandas as pd

df = pd.DataFrame({
    "이름": ["민수", "지아", "하준"],
    "점수": [80, 95, 70]
})
print(df)

DataFrame은 표 형태 데이터입니다. 행과 열을 가진 데이터 분석의 기본 구조입니다.

코드 3. 기초 통계

print(df["점수"].mean())
print(df["점수"].max())

mean()은 평균, max()는 최댓값을 구합니다.

코드 4. 결측값 처리

df["점수"] = df["점수"].fillna(df["점수"].mean())

fillna()는 비어 있는 값을 채웁니다. 여기서는 평균값으로 결측값을 대체합니다.

코드 5. 문자열 공백 제거

df["지역"] = df["지역"].str.strip()

str.strip()은 문자열 양쪽 공백을 제거하여 같은 지역명이 다르게 인식되는 문제를 줄입니다.

코드 6. 그룹별 집계

summary = df.groupby("지역")["매출"].sum()
print(summary)

groupby()는 특정 기준으로 데이터를 묶고, sum()으로 그룹별 합계를 계산합니다.

코드 7. Matplotlib 막대그래프

import matplotlib.pyplot as plt

plt.bar(["A", "B", "C"], [10, 20, 15])
plt.title("Bar Chart")
plt.show()

막대그래프는 범주별 크기를 비교할 때 사용합니다.

코드 8. Seaborn 산점도

import seaborn as sns

sns.scatterplot(data=df, x="공부시간", y="점수")
plt.show()

산점도는 두 변수 사이의 관계를 확인할 때 사용합니다.

코드 9. train_test_split

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(
    X, y, random_state=42
)

데이터를 학습용과 테스트용으로 나누어 모델 성능을 공정하게 평가합니다.

코드 10. 의사결정나무 분류

from sklearn.tree import DecisionTreeClassifier

model = DecisionTreeClassifier(random_state=42)
model.fit(X_train, y_train)
accuracy = model.score(X_test, y_test)
print(accuracy)

DecisionTreeClassifier는 분류 문제에 사용하는 머신러닝 모델입니다. score()는 테스트 정확도를 계산합니다.

🧪 프로젝트

1. 학생 성적 분석

import pandas as pd

df = pd.DataFrame({
    "이름": ["민수", "지아", "하준"],
    "국어": [80, 90, 70],
    "수학": [85, 95, 75]
})
df["평균"] = df[["국어", "수학"]].mean(axis=1)
print(df)
print("전체 평균:", df["평균"].mean())

2. 매출 분석

import pandas as pd
import matplotlib.pyplot as plt

data = {"지역": ["서울", "서울", "부산", "부산"], "매출": [120, 150, 80, 100]}
df = pd.DataFrame(data)
summary = df.groupby("지역")["매출"].sum()

summary.plot(kind="bar")
plt.title("지역별 매출 합계")
plt.show()

3. 붓꽃 데이터 분류

from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier

X, y = load_iris(return_X_y=True)

X_train, X_test, y_train, y_test = train_test_split(
    X, y, random_state=42
)

model = DecisionTreeClassifier(random_state=42)
model.fit(X_train, y_train)

print("정확도:", model.score(X_test, y_test))

🌟 수업마무리하기

데이터 분석과 AI는 단순히 라이브러리를 호출하는 것이 아니라, 데이터를 이해하고 문제를 정의하며 결과를 해석하는 과정입니다.

  • NumPy는 빠른 수치 계산을 돕는다.
  • Pandas는 표 데이터를 분석한다.
  • 전처리는 분석 품질을 높인다.
  • 시각화는 데이터 패턴을 쉽게 보여준다.
  • 기초 통계는 데이터를 요약한다.
  • 머신러닝은 데이터에서 규칙을 학습한다.

핵심 정리: 좋은 AI 프로젝트는 좋은 데이터 이해와 전처리에서 시작됩니다.
