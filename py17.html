01. Flask 기본 웹 서버 만들기

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello Flask!"

if __name__ == "__main__":
    app.run(debug=True)

🔍 초보자용 한 줄씩 상세 설명

줄코드꼼꼼한 설명
1from flask import FlaskFlask 라이브러리에서 Flask 클래스를 가져옵니다. 이 클래스가 웹 서버 앱을 만드는 핵심 도구입니다.
2app = Flask(__name__)Flask 웹 애플리케이션 객체를 만듭니다. __name__은 현재 파일 이름 정보를 Flask에 알려 주어, 템플릿이나 정적 파일 위치를 찾는 기준이 됩니다.
3@app.route("/")사용자가 웹 브라우저에서 기본 주소 / 로 접속했을 때 바로 아래의 home() 함수를 실행하라는 뜻입니다.
4def home():home이라는 함수를 정의합니다. Flask에서는 이런 함수를 보통 view 함수라고 부릅니다.
5return "Hello Flask!"브라우저에 보낼 응답 내용을 반환합니다. 이 경우 화면에 Hello Flask!가 보입니다.
6if __name__ == "__main__":이 파일을 직접 실행했을 때만 아래 코드를 실행하라는 조건문입니다. 다른 파일에서 import될 때는 실행되지 않게 합니다.
7app.run(debug=True)Flask 개발용 서버를 실행합니다. debug=True는 코드 수정 시 자동 재시작과 오류 화면 표시를 도와줍니다.

03. Flask URL 변수 사용

from flask import Flask

app = Flask(__name__)

@app.route("/user/<name>")
def user(name):
    return f"<h1>{name}님 환영합니다!</h1>"

if __name__ == "__main__":
    app.run(debug=True)

3@app.route("/user/<name>")URL에서 /user/ 뒤에 오는 값을 name이라는 변수로 받아 오도록 설정합니다. 예를 들어 /user/Kim이면 name은 Kim입니다.

04. Flask GET 요청 파라미터 처리

from flask import Flask, request

app = Flask(__name__)

@app.route("/search")
def search():
    keyword = request.args.get("q", "검색어 없음")
    return f"<h1>검색어: {keyword}</h1>"

if __name__ == "__main__":
    app.run(debug=True)

🔍 초보자용 한 줄씩 상세 설명

줄코드꼼꼼한 설명
1from flask import Flask, requestFlask 앱을 만들기 위한 Flask와, 사용자의 요청 정보를 읽기 위한 request를 가져옵니다.
2app = Flask(__name__)Flask 웹 애플리케이션 객체를 생성합니다.
3@app.route("/search")/search 주소로 요청이 들어오면 search() 함수를 실행하도록 연결합니다.
4def search():검색 요청을 처리할 search 함수를 정의합니다.
5keyword = request.args.get("q", "검색어 없음")URL 뒤에 붙는 ?q=값 형태의 GET 파라미터에서 q 값을 읽습니다. q가 없으면 검색어 없음이라는 기본값을 사용합니다.
6return f"<h1>검색어: {keyword}</h1>"읽어 온 keyword 값을 HTML 제목 형태로 브라우저에 반환합니다.

05. Flask POST 폼 데이터 처리

from flask import Flask, request

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def form():
    if request.method == "POST":
        name = request.form.get("name")
        return f"<h1>{name}님 제출 완료!</h1>"

    return """
    <form method="post">
        <input name="name" placeholder="이름 입력">
        <button type="submit">제출</button>
    </form>
    """

if __name__ == "__main__":
    app.run(debug=True)

🔍 초보자용 한 줄씩 상세 설명

줄코드꼼꼼한 설명
1from flask import Flask, requestFlask 앱 생성 도구와 요청 데이터를 읽는 request 객체를 가져옵니다.
2app = Flask(__name__)Flask 웹 애플리케이션 객체를 만듭니다.
3@app.route("/", methods=["GET", "POST"])기본 주소 / 에서 GET 요청과 POST 요청을 모두 받을 수 있게 설정합니다.
4def form():폼 화면을 보여주거나 제출 데이터를 처리할 form 함수를 정의합니다.
5if request.method == "POST":현재 요청 방식이 POST인지 확인합니다. 사용자가 폼을 제출하면 보통 POST 요청이 됩니다.
6name = request.form.get("name")폼에서 name이라는 이름을 가진 입력칸의 값을 가져옵니다.
7return f"<h1>{name}님 제출 완료!</h1>"입력받은 이름을 넣어 제출 완료 메시지를 HTML로 반환합니다.
8return """POST가 아닌 경우, 즉 처음 접속한 GET 요청일 때 보여줄 HTML 폼을 반환하기 시작합니다.
9<form method="post">HTML 폼을 만듭니다. method='post'는 제출 시 POST 방식으로 서버에 데이터를 보내겠다는 뜻입니다.
10<input name="name" placeholder="이름 입력">사용자가 이름을 입력할 수 있는 입력칸입니다. name='name'이므로 서버에서 request.form.get('name')으로 읽을 수 있습니다.
11<button type="submit">제출</button>폼을 제출하는 버튼입니다.
12</form>HTML 폼 태그를 닫습니다.
13"""여러 줄 HTML 문자열을 끝냅니다.
14if __name__ == "__main__":직접 실행할 때만 서버 실행 코드가 작동하도록 합니다.
15app.run(debug=True)Flask 개발 서버를 실행합니다.

08. Django HTML 템플릿 렌더링

# views.py
from django.shortcuts import render

def home(request):
    context = {
        "title": "Django 수업",
        "message": "템플릿으로 HTML을 렌더링합니다."

    return render(request, "home.html", context)

<h1>{{ title }}</h1>

10. Django 간단한 게시글 목록 페이지

# views.py
from django.shortcuts import render

def post_list(request):
    posts = [
        {"title": "첫 번째 글", "author": "Kim"},
        {"title": "두 번째 글", "author": "Lee"},
        {"title": "세 번째 글", "author": "Park"},
    ]
    return render(request, "post_list.html", {"posts": posts})

# templates/post_list.html
<h1>게시글 목록</h1>
<ul>
{% for post in posts %}
    <li>{{ post.title }} - {{ post.author }}</li>
{% endfor %}
</ul>
