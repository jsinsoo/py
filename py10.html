🧠 핵심 개념

1. 웹 개발

웹 개발은 사용자가 브라우저에서 접근할 수 있는 서비스를 만드는 과정입니다. Flask는 가볍고 빠른 웹앱에, Django는 큰 규모의 웹 서비스에 자주 사용됩니다.

2. API 활용

API는 프로그램끼리 데이터를 주고받는 약속입니다. 공공데이터포털 API를 활용하면 날씨, 미세먼지, 버스 도착 정보 등을 가져올 수 있습니다.

3. 웹 크롤링

크롤링은 웹 페이지에서 필요한 정보를 수집하는 과정입니다. robots.txt, 이용약관, 저작권을 지켜야 합니다.

4. 자동화와 GUI

자동화는 반복 작업을 프로그램이 대신하게 하는 기술이고, GUI는 사용자가 버튼과 입력창으로 프로그램을 쉽게 조작하도록 만드는 화면입니다.

5. 데이터·AI 프로젝트

데이터 프로젝트는 수집·전처리·분석·시각화로 구성되고, AI 프로젝트는 여기에 모델 학습과 예측 기능이 추가됩니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. Flask 기본 서버

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello Flask"

app.run(debug=True)

Flask는 간단한 웹 서버를 빠르게 만들 수 있습니다. "/" 주소로 접속하면 Hello Flask가 보입니다.

코드 2. Flask HTML 반환

@app.route("/hello")
def hello():
    return "<h1>안녕하세요</h1>"

Flask는 문자열뿐 아니라 HTML 코드도 응답으로 보낼 수 있습니다.

코드 3. Django 개념 구조

# Django 기본 흐름
# URL 요청 → view 함수 → template 렌더링 → HTML 응답

Django는 URL, view, template, model 구조로 웹 서비스를 체계적으로 구성합니다.

코드 4. API 요청

import requests

url = "https://api.example.com/data"
response = requests.get(url)
print(response.status_code)

requests.get()은 API 서버에 데이터를 요청합니다. status_code 200은 일반적으로 성공을 의미합니다.

코드 5. JSON 응답 처리

data = response.json()
print(data)

API 응답이 JSON 형식이면 response.json()으로 파이썬 딕셔너리처럼 다룰 수 있습니다.

코드 6. 공공데이터포털 API 기본 형태

params = {
    "serviceKey": "인증키",
    "pageNo": 1,
    "numOfRows": 10,
    "dataType": "JSON"
}

공공데이터포털 API는 보통 인증키와 요청 파라미터를 함께 전달합니다. 실제 수업에서는 인증키를 공개하지 않도록 주의합니다.

코드 7. Beautiful Soup 파싱

from bs4 import BeautifulSoup

html = "<h1>Python</h1>"
soup = BeautifulSoup(html, "html.parser")
print(soup.h1.text)

Beautiful Soup은 HTML 문서에서 원하는 태그와 텍스트를 추출할 때 사용합니다.

코드 8. Selenium 자동화 개념

from selenium import webdriver

driver = webdriver.Chrome()
driver.get("https://example.com")
print(driver.title)
driver.quit()

Selenium은 실제 브라우저를 조작해 클릭, 입력, 페이지 이동 같은 작업을 자동화할 수 있습니다.

코드 9. Tkinter GUI

import tkinter as tk

root = tk.Tk()
root.title("파이썬 GUI")
label = tk.Label(root, text="Hello Tkinter")
label.pack()
root.mainloop()

Tkinter는 파이썬 기본 GUI 라이브러리입니다. 창, 라벨, 버튼, 입력창 등을 만들 수 있습니다.

코드 10. 챗봇 기본 구조

def chatbot(msg):
    if "안녕" in msg:
        return "안녕하세요!"
    return "무슨 뜻인지 더 알려주세요."

print(chatbot("안녕"))

간단한 규칙 기반 챗봇입니다. 이후 AI 모델과 연결하면 더 지능적인 챗봇으로 확장할 수 있습니다.

🌟 수업마무리하기

고급 프로젝트 단계에서는 문법보다 중요한 것이 있습니다. 바로 문제를 실제로 해결하는 완성형 결과물을 만드는 것입니다.

  • 웹 개발은 사용자에게 서비스를 제공한다.
  • API는 외부 데이터를 연결한다.
  • 크롤링은 웹 정보를 수집하지만 윤리와 규칙을 지켜야 한다.
  • 자동화는 반복 작업을 줄인다.
  • GUI는 사용 편의성을 높인다.
  • 데이터·AI 프로젝트는 분석과 예측으로 의사결정을 돕는다.

핵심 정리: 실무형 개발자는 코드를 작성하는 사람을 넘어, 문제를 발견하고 해결 가능한 서비스로 구현하는 사람입니다.
