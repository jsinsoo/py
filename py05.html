실전 프로그램은 실행이 끝나도 데이터를 보관해야 합니다. 메모장, 성적 저장 프로그램, 로그 기록 시스템은 모두 파일 처리가 필요합니다. 또한 사용자가 잘못 입력하거나 파일이 없을 때 프로그램이 멈추지 않도록 예외 처리가 필요합니다.

🧠 핵심 개념

1. 파일 처리

파일 처리는 프로그램 실행 결과를 저장하거나, 저장된 데이터를 다시 읽어오는 과정입니다.

2. with 문

with 문은 파일을 열고 작업이 끝나면 자동으로 닫아 주므로 안전하고 깔끔한 파일 처리 방식입니다.

3. CSV

CSV는 쉼표로 구분된 표 형식 데이터입니다. 성적표, 매출표, 출석부 등에 자주 사용됩니다.

4. JSON

JSON은 키와 값 형태로 데이터를 저장하는 형식입니다. 웹 API와 설정 파일에서 많이 사용됩니다.

5. 예외 처리

try-except-finally는 오류가 발생해도 프로그램이 갑자기 멈추지 않도록 제어하는 구조입니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. 파일 쓰기

f = open("memo.txt", "w", encoding="utf-8")
f.write("안녕하세요")
f.close()

open()으로 파일을 쓰기 모드로 열고 write()로 내용을 저장합니다. close()로 파일을 닫습니다.

코드 2. with 문으로 파일 쓰기

with open("memo.txt", "w", encoding="utf-8") as f:
    f.write("파이썬 파일 처리")

with 문을 사용하면 파일을 자동으로 닫아 주므로 close()를 직접 쓰지 않아도 됩니다.

코드 3. 파일 읽기

with open("memo.txt", "r", encoding="utf-8") as f:
    text = f.read()
print(text)

read()는 파일 전체 내용을 문자열로 읽어옵니다.

코드 4. 여러 줄 쓰기

lines = ["국어 90\n", "수학 95\n"]
with open("score.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)

writelines()는 문자열 리스트를 파일에 연속으로 기록합니다.

코드 5. 한 줄씩 읽기

with open("score.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

파일 객체를 반복하면 한 줄씩 읽을 수 있습니다. strip()은 줄바꿈과 양쪽 공백을 제거합니다.

코드 6. CSV 쓰기

import csv

with open("students.csv", "w", newline="", encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerow(["이름", "점수"])
    writer.writerow(["민수", 90])

csv.writer를 사용하면 표 형태 데이터를 CSV 파일로 저장할 수 있습니다.

코드 7. CSV 읽기

import csv

with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)

csv.reader는 CSV 파일의 각 행을 리스트 형태로 읽어옵니다.

코드 8. JSON 저장

import json

data = {"name": "Kim", "score": 95}
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False)

json.dump()는 딕셔너리 데이터를 JSON 파일로 저장합니다. ensure_ascii=False는 한글을 깨지지 않게 저장합니다.

코드 9. try-except

try:
    n = int("abc")
except ValueError:
    print("숫자로 변환할 수 없습니다.")

int("abc")는 오류가 발생하지만 except가 이를 처리하여 프로그램이 멈추지 않습니다.

코드 10. finally

try:
    print(10 / 2)
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")
finally:
    print("처리 완료")

finally는 오류 발생 여부와 관계없이 마지막에 반드시 실행됩니다.

🌟 수업마무리하기

파일 처리와 예외 처리는 실전 프로그램의 기본입니다. 데이터를 저장하지 못하면 프로그램은 일회성 계산기에 머물고, 오류를 처리하지 못하면 실제 사용자가 쓰기 어렵습니다.

  • 파일 쓰기: 데이터를 저장한다.
  • 파일 읽기: 저장된 데이터를 불러온다.
  • with 문: 파일을 안전하게 다룬다.
  • CSV: 표 데이터를 저장한다.
  • JSON: 구조화된 데이터를 저장한다.
  • try-except-finally: 오류에 강한 프로그램을 만든다.

핵심 정리: 실전 프로그램은 데이터를 보존하고, 예외 상황에서도 안정적으로 동작해야 합니다.
