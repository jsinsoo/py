필요한 항목만 골라 출력·저장·시각화합니다.

01. 기상청 단기예보 조회

기상청 단기예보 API에 요청을 보내 특정 격자 좌표의 예보 데이터를 JSON으로 가져오는 기본 구조입니다.

import requests

service_key = "공공데이터포털_인증키"

url = "http://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getVilageFcst"

params = {
    "serviceKey": service_key,
    "pageNo": 1,
    "numOfRows": 100,
    "dataType": "JSON",
    "base_date": "20260509",
    "base_time": "0500",
    "nx": 63,
    "ny": 89
}

res = requests.get(url, params=params)
data = res.json()

print(data)

핵심 포인트

  • serviceKey: 공공데이터포털에서 발급받은 인증키를 입력합니다.
  • url: 사용할 API의 요청 주소입니다.
  • params: API에 전달할 필수·선택 파라미터입니다.
  • requests.get(): 서버에 요청을 보내고 응답을 받습니다.
  • res.json(): JSON 응답을 파이썬 딕셔너리처럼 다룰 수 있게 변환합니다.

02. 초단기실황 조회

초단기실황 API에서 현재 기온, 습도, 강수량 등 관측값을 가져와 category와 obsrValue를 출력합니다.

import requests

service_key = "공공데이터포털_인증키"

url = "http://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getUltraSrtNcst"

params = {
    "serviceKey": service_key,
    "pageNo": 1,
    "numOfRows": 50,
    "dataType": "JSON",
    "base_date": "20260509",
    "base_time": "0600",
    "nx": 63,
    "ny": 89
}

res = requests.get(url, params=params)
data = res.json()

items = data["response"]["body"]["items"]["item"]

for item in items:
    print(item["category"], item["obsrValue"])

핵심 포인트

  • serviceKey: 공공데이터포털에서 발급받은 인증키를 입력합니다.
  • url: 사용할 API의 요청 주소입니다.
  • params: API에 전달할 필수·선택 파라미터입니다.
  • requests.get(): 서버에 요청을 보내고 응답을 받습니다.
  • res.json(): JSON 응답을 파이썬 딕셔너리처럼 다룰 수 있게 변환합니다.

08. 공공데이터 응답 상태 확인 디버깅 코드

import requests

    "pageNo": 1,
    "numOfRows": 10,
    "dataType": "JSON",
    "base_date": "20260509",
    "base_time": "0600",
    "nx": 63,
    "ny": 89
}

print("요청 URL:")
print(res.url)

print("상태 코드:")
print(res.status_code)

print("응답 앞부분:")

09. JSONDecodeError 방지 코드

응답이 JSON이 아닐 때 발생하는 JSONDecodeError를 처리하여 프로그램이 멈추지 않게 합니다.

import requests
import json

service_key = "공공데이터포털_인증키"

url = "http://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getUltraSrtNcst"

params = {
    "serviceKey": service_key,
    "pageNo": 1,
    "numOfRows": 10,
    "dataType": "JSON",
    "base_date": "20260509",
    "base_time": "0600",
    "nx": 63,
    "ny": 89
}

res = requests.get(url, params=params)

try:
    data = res.json()
    print(data)
except json.JSONDecodeError:
    print("JSON 변환 실패")
    print("응답 내용:")
    print(res.text[:500])

핵심 포인트

  • serviceKey: 공공데이터포털에서 발급받은 인증키를 입력합니다.
  • url: 사용할 API의 요청 주소입니다.
  • params: API에 전달할 필수·선택 파라미터입니다.
  • requests.get(): 서버에 요청을 보내고 응답을 받습니다.
  • res.json(): JSON 응답을 파이썬 딕셔너리처럼 다룰 수 있게 변환합니다.

🏁 최종 정리

공공데이터포털 API 선택
→ 인증키 발급
→ 요청 URL 확인
→ 필수 파라미터 입력
→ requests.get()

오류 해결 핵심

  • 응답이 비어 있으면 res.url을 출력해 실제 요청 URL을 확인합니다.
  • 상태 코드가 403이면 인증키, 접근 권한, URL, 파라미터를 확인합니다.
  • JSON 변환 오류가 나면 res.text[:500]으로 응답 앞부분을 먼저 확인합니다.
  • 기상청 API는 base_date, base_time, nx, ny가 매우 중요합니다.
