입력처리출력저장계산

파이썬 프로그램은 대부분 이 흐름으로 움직입니다.

🚀 동기유발 코드

print("안녕하세요, 파이썬!")

이 한 줄은 컴퓨터 화면에 문장을 출력합니다. 프로그램의 시작은 명령을 정확히 쓰는 것입니다.

🎯 학습목표

파이썬 기초 문법을 활용하여 간단한 문제를 해결할 수 있다.

📘 지식 목표

  • print(), input()의 역할을 설명한다.
  • 변수와 자료형의 의미를 이해한다.
  • 산술·비교·논리 연산자를 구분한다.

🛠 기능 목표

  • 입력과 출력을 이용한 프로그램을 작성한다.
  • int, float, str, bool 자료형을 구분한다.
  • 형 변환을 이용하여 계산 프로그램을 만든다.

💡 태도 목표

  • 오류를 수정하며 끈기 있게 탐구한다.
  • 코드의 실행 결과를 논리적으로 예측한다.
  • 생활 문제를 프로그래밍으로 해결하려는 태도를 가진다.

💻 개념기반 학습코드 10개

각 코드는 파이썬 기초 문법의 핵심 개념과 연결됩니다.

🧪 실습 예제

계산기, BMI 계산기, 성적 평균 계산기로 기초 문법을 실제 문제에 적용합니다.

🧮 계산기 만들기

a = int(input("첫 번째 수: "))
b = int(input("두 번째 수: "))
print("합계:", a + b)

input()은 문자열을 반환하므로 int()로 정수 변환 후 계산합니다.

⚖️ BMI 계산기

weight = float(input("몸무게 kg: "))
height = float(input("키 m: "))
bmi = weight / (height * height)
print("BMI:", bmi)

소수 계산이 필요하므로 float()를 사용합니다.

🏫 성적 평균 계산

kor = int(input("국어: "))
math = int(input("수학: "))
eng = int(input("영어: "))
avg = (kor + math + eng) / 3
print("평균:", avg)

여러 값을 변수에 저장하고 산술 연산으로 평균을 구합니다.
