자료구조와 알고리즘은 단순히 코드를 외우는 단원이 아니라, 문제를 어떤 방식으로 저장하고 어떤 절차로 해결할 것인가를 설계하는 단원입니다.

📚 스택

책을 쌓듯이 마지막에 들어온 데이터가 먼저 나가는 구조입니다.

🚶 큐

줄 서기처럼 먼저 들어온 데이터가 먼저 나가는 구조입니다.

🔍 탐색

많은 데이터 속에서 원하는 값을 빠르게 찾는 방법입니다.

⚡ 시간복잡도

데이터가 커질 때 알고리즘이 얼마나 빨리 동작하는지 판단합니다.

기능 목표버블 정렬, 선택 정렬, 병합 정렬, 이진 탐색, DFS, BFS, 유클리드 알고리즘을 파이썬으로 구현할 수 있다.
태도 목표문제의 조건과 데이터 규모에 맞는 효율적인 해결 전략을 선택하려는 태도를 기른다.

🧠 핵심 개념

1. 자료구조

자료구조는 데이터를 저장하고 관리하는 방식입니다. 같은 데이터라도 저장 방식에 따라 문제 해결 속도와 코드 구조가 달라집니다.

2. 알고리즘

알고리즘은 문제를 해결하기 위한 명확한 절차입니다. 정렬, 탐색, 최단 경로, 최대공약수 계산 등이 대표적입니다.

3. 시간복잡도

시간복잡도는 입력 크기 n이 커질 때 실행 시간이 어떻게 증가하는지 나타냅니다. 예: O(1), O(log n), O(n), O(n²).

4. 재귀

재귀는 함수가 자기 자신을 다시 호출하는 구조입니다. 문제를 작은 문제로 나누어 해결할 때 사용합니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. 스택

stack = []
stack.append(10)
stack.append(20)
print(stack.pop())

스택은 마지막에 들어온 값이 먼저 나갑니다. 20이 마지막에 들어왔으므로 pop() 결과는 20입니다.

코드 2. 큐

from collections import deque
q = deque()
q.append("A")
q.append("B")
print(q.popleft())

큐는 먼저 들어온 값이 먼저 나갑니다. A가 먼저 들어왔으므로 popleft() 결과는 A입니다.

코드 3. 해시

score = {"Kim": 90, "Lee": 85}
print(score["Kim"])

딕셔너리는 해시 구조를 활용하여 키로 값을 빠르게 찾습니다.

코드 4. 버블 정렬

a = [3, 1, 2]
for i in range(len(a)):
    for j in range(len(a)-1):
        if a[j] > a[j+1]:
            a[j], a[j+1] = a[j+1], a[j]
print(a)

인접한 두 값을 비교하여 큰 값을 뒤로 보내는 정렬입니다.

코드 5. 선택 정렬

a = [3, 1, 2]
for i in range(len(a)):
    m = i
    for j in range(i+1, len(a)):
        if a[j] < a[m]:
            m = j
    a[i], a[m] = a[m], a[i]
print(a)

가장 작은 값을 찾아 앞쪽에 배치하는 방식입니다.

코드 6. 이진 탐색

a = [1, 3, 5, 7, 9]
target = 7
left, right = 0, len(a)-1
while left <= right:
    mid = (left + right) // 2
    if a[mid] == target:
        print(mid)
        break
    elif a[mid] < target:
        left = mid + 1
    else:
        right = mid - 1

정렬된 데이터에서 중간값을 기준으로 탐색 범위를 절반씩 줄입니다.

코드 7. 재귀 팩토리얼

def fact(n):
    if n == 1:
        return 1
    return n * fact(n-1)

print(fact(5))

5! = 5×4×3×2×1 = 120입니다. 재귀에는 종료 조건이 반드시 필요합니다.

코드 8. DFS

graph = {1:[2,3], 2:[4], 3:[], 4:[]}
visited = []

def dfs(v):
    visited.append(v)
    for nxt in graph[v]:
        if nxt not in visited:
            dfs(nxt)

dfs(1)
print(visited)

DFS는 깊이 우선 탐색입니다. 한 방향으로 깊게 들어간 뒤 되돌아옵니다.

코드 9. BFS

from collections import deque
graph = {1:[2,3], 2:[4], 3:[], 4:[]}
visited = []
q = deque([1])

while q:
    v = q.popleft()
    if v not in visited:
        visited.append(v)
        q.extend(graph[v])
print(visited)

BFS는 너비 우선 탐색입니다. 가까운 정점부터 차례로 방문합니다.

코드 10. 유클리드 알고리즘

def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

print(gcd(48, 18))

두 수의 최대공약수를 빠르게 구하는 대표 알고리즘입니다. 결과는 6입니다.

알고리즘핵심 아이디어대표 활용

🌟 수업마무리하기

자료구조와 알고리즘은 컴퓨터가 문제를 더 빠르고 정확하게 해결하도록 돕는 핵심 도구입니다.

  • 스택은 마지막 값이 먼저 나간다.
  • 큐는 먼저 들어온 값이 먼저 나간다.
  • 해시는 키로 값을 빠르게 찾는다.
  • 정렬은 데이터를 일정한 순서로 재배치한다.
  • 탐색은 원하는 값을 찾는 과정이다.
  • 재귀는 문제를 작은 문제로 나눈다.
  • 시간복잡도는 알고리즘의 효율성을 비교한다.

핵심 정리: 좋은 알고리즘은 정확할 뿐만 아니라 입력이 커져도 효율적으로 동작해야 합니다.
