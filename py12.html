01. 웹페이지 제목 가져오기

import requests
from bs4 import BeautifulSoup

url = "https://example.com"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

print("웹페이지 제목:", soup.title.text)

줄코드상세 설명
1import requests웹 서버에 HTTP 요청을 보내기 위해 requests 라이브러리를 불러옵니다.
2from bs4 import BeautifulSoupHTML 문서를 분석하기 위해 BeautifulSoup 클래스를 불러옵니다.
3url = "https://example.com"크롤링할 웹페이지 주소를 문자열로 저장합니다.
4res = requests.get(url)지정한 URL에 GET 요청을 보내고 서버 응답을 res 변수에 저장합니다.
5soup = BeautifulSoup(res.text, "html.parser")응답으로 받은 HTML 문자열을 BeautifulSoup 객체로 변환합니다.
6print("웹페이지 제목:", soup.title.text)HTML의 title 태그 안에 있는 텍스트를 출력합니다.

02. 뉴스 제목 크롤링

from bs4 import BeautifulSoup

06. 크롤링 결과 CSV 저장

import requests
from bs4 import BeautifulSoup
import csv

url = "https://news.ycombinator.com/"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")
titles = soup.find_all("span", class_="titleline")

with open("news.csv", "w", newline="", encoding="utf-8-sig") as f:
    writer = csv.writer(f)
    writer.writerow(["번호", "제목"])

    for i, title in enumerate(titles[:10], 1):
        writer.writerow([i, title.text])

🔍 한 줄씩 상세 설명

줄코드상세 설명
1import requests뉴스 페이지 HTML을 요청하기 위해 requests를 불러옵니다.
2from bs4 import BeautifulSoupHTML에서 뉴스 제목 태그를 찾기 위해 BeautifulSoup을 불러옵니다.
3import csv수집한 데이터를 CSV 파일로 저장하기 위해 csv 모듈을 불러옵니다.
4url = "https://news.ycombinator.com/"수집할 뉴스 페이지 주소를 저장합니다.
5res = requests.get(url)웹페이지에 요청을 보내 응답을 받습니다.
6soup = BeautifulSoup(res.text, "html.parser")응답 HTML을 BeautifulSoup 객체로 변환합니다.
7titles = soup.find_all("span", class_="titleline")뉴스 제목이 들어 있는 span.titleline 요소들을 모두 찾습니다.
8with open("news.csv", "w", newline="", encoding="utf-8-sig") as f:news.csv 파일을 쓰기 모드로 열고, 한글 호환을 위해 utf-8-sig 인코딩을 사용합니다.
9writer = csv.writer(f)CSV 파일에 행 단위로 기록할 writer 객체를 만듭니다.
10writer.writerow(["번호", "제목"])CSV 첫 줄에 열 제목을 기록합니다.
11for i, title in enumerate(titles[:10], 1):앞 10개의 뉴스 제목에 1부터 번호를 붙여 반복합니다.
12writer.writerow([i, title.text])번호와 제목 텍스트를 CSV 파일에 한 줄씩 저장합니다.

08. Selenium 로그인 자동화 구조

from selenium import webdriver
from selenium.webdriver.common.by import By
import time

driver = webdriver.Chrome()

driver.get("https://example.com/login")

driver.find_element(By.NAME, "username").send_keys("myid")
driver.find_element(By.NAME, "password").send_keys("mypassword")

time.sleep(2)

driver.quit()

🔍 한 줄씩 상세 설명

줄코드상세 설명
1from selenium import webdriverSelenium으로 브라우저를 실행하기 위해 webdriver를 불러옵니다.
2from selenium.webdriver.common.by import ByHTML 요소를 찾는 기준을 지정하기 위해 By를 불러옵니다.
3import time실행 중 잠시 기다리기 위해 time 모듈을 불러옵니다.
4driver = webdriver.Chrome()크롬 브라우저를 자동화 제어 대상으로 실행합니다.
5driver.get("https://example.com/login")브라우저로 로그인 페이지 주소에 접속합니다.
6driver.find_element(By.NAME, "username").send_keys("myid")name 속성이 username인 입력창을 찾아 myid를 입력합니다.
7driver.find_element(By.NAME, "password").send_keys("mypassword")name 속성이 password인 입력창을 찾아 비밀번호 문자열을 입력합니다.
8time.sleep(2)브라우저 동작을 확인하기 위해 2초 동안 기다립니다.
9driver.quit()브라우저를 종료하고 Selenium 세션을 정리합니다.

09. 무한 스크롤 페이지 크롤링

from selenium import webdriver
import time

driver = webdriver.Chrome()
driver.get("https://example.com")

for _ in range(5):
    driver.execute_script(
        "window.scrollTo(0, document.body.scrollHeight);"
    )
    time.sleep(2)

print(driver.page_source)

driver.quit()

🔍 한 줄씩 상세 설명

줄코드상세 설명
1from selenium import webdriver브라우저 자동화를 위해 Selenium webdriver를 불러옵니다.
2import time스크롤 후 새 내용이 로딩될 시간을 주기 위해 time 모듈을 불러옵니다.
3driver = webdriver.Chrome()크롬 브라우저를 실행합니다.
4driver.get("https://example.com")크롤링할 페이지로 이동합니다.
5for _ in range(5):스크롤 동작을 5번 반복합니다.
6driver.execute_script(브라우저에서 JavaScript 코드를 실행합니다.
7"window.scrollTo(0, document.body.scrollHeight);"페이지 맨 아래까지 스크롤하라는 JavaScript 명령입니다.
8time.sleep(2)새 콘텐츠가 로딩될 수 있도록 2초 기다립니다.
9print(driver.page_source)현재 브라우저에 로딩된 전체 HTML 소스를 출력합니다.

10. 크롤링 → Pandas 분석

import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "https://books.toscrape.com/"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

titles = soup.find_all("h3")

data = []

for t in titles:
    data.append(t.text.strip())

df = pd.DataFrame(data, columns=["책 제목"])

print(df)

🔍 한 줄씩 상세 설명

줄코드상세 설명
1import requests책 목록 페이지 HTML을 요청하기 위해 requests를 불러옵니다.
2from bs4 import BeautifulSoupHTML에서 책 제목 태그를 찾기 위해 BeautifulSoup을 불러옵니다.
3import pandas as pd수집한 데이터를 표 형태 DataFrame으로 만들기 위해 pandas를 pd라는 이름으로 불러옵니다.
4url = "https://books.toscrape.com/"크롤링할 연습용 책 사이트 주소를 저장합니다.
5res = requests.get(url)사이트에 GET 요청을 보내 HTML 응답을 받습니다.
6soup = BeautifulSoup(res.text, "html.parser")HTML 문자열을 BeautifulSoup 객체로 변환합니다.
7titles = soup.find_all("h3")책 제목이 들어 있는 h3 태그들을 모두 찾습니다.
8data = []수집한 책 제목을 저장할 빈 리스트를 만듭니다.
9for t in titles:찾은 h3 태그를 하나씩 반복합니다.
10data.append(t.text.strip())태그의 텍스트에서 양쪽 공백을 제거한 뒤 data 리스트에 추가합니다.
11df = pd.DataFrame(data, columns=["책 제목"])수집한 리스트를 책 제목이라는 열을 가진 DataFrame으로 변환합니다.
12print(df)완성된 DataFrame을 출력합니다.
