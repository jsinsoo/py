🎯 학습목표

영역목표
지식 목표리스트, 튜플, 딕셔너리, 집합, 문자열의 특징과 차이를 설명할 수 있다.
기능 목표자료구조를 활용하여 성적 관리, 단어 빈도 분석, 중복 제거, 슬라이싱 코드를 작성할 수 있다.
태도 목표문제 상황에 맞는 자료구조를 선택하여 효율적으로 데이터를 관리하려는 태도를 기른다.

🧠 핵심 개념

1. 리스트(list)

여러 값을 순서대로 저장하며, 추가·수정·삭제가 가능합니다. 예: 점수 목록, 이름 목록.

2. 튜플(tuple)

순서가 있지만 한 번 만든 뒤 값을 바꿀 수 없습니다. 예: 좌표, 고정 설정값.

3. 딕셔너리(dict)

키와 값의 쌍으로 데이터를 저장합니다. 예: 학생 이름 → 점수, 단어 → 빈도.

4. 집합(set)

중복을 허용하지 않는 자료구조입니다. 예: 중복 제거, 공통 원소 찾기.

5. 문자열과 슬라이싱

문자열도 문자들의 순서 있는 자료로 볼 수 있으며, 인덱스와 슬라이싱으로 일부를 꺼낼 수 있습니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. 리스트 생성과 인덱싱

scores = [90, 85, 70]
print(scores[0])

리스트는 0번부터 인덱스를 셉니다. scores[0]은 첫 번째 값 90입니다.

코드 2. 리스트 값 추가

names = ["민수", "지아"]
names.append("하준")
print(names)

append()는 리스트 맨 뒤에 새 값을 추가합니다.

코드 3. 리스트 합계와 평균

scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(avg)

sum()은 합계, len()은 개수를 구합니다. 평균은 합계 ÷ 개수입니다.

코드 4. 튜플

point = (3, 4)
print(point[0] + point[1])

튜플은 순서가 있는 고정 데이터입니다. 좌표처럼 바뀌면 안 되는 값에 적합합니다.

코드 5. 딕셔너리 생성

student = {"name": "Kim", "score": 95}
print(student["score"])

딕셔너리는 키를 이용해 값을 꺼냅니다. "score" 키의 값은 95입니다.

코드 6. 딕셔너리 값 수정

scores = {"국어": 80, "수학": 90}
scores["수학"] = 100
print(scores)

키를 이용해 특정 값을 수정할 수 있습니다.

코드 7. 집합으로 중복 제거

nums = [1, 2, 2, 3, 3, 3]
unique = set(nums)
print(unique)

set은 중복을 제거합니다. 단, 출력 순서는 환경에 따라 달라질 수 있습니다.

코드 8. 문자열 인덱싱

word = "Python"
print(word[0])
print(word[-1])

word[0]은 첫 글자 P, word[-1]은 마지막 글자 n입니다.

코드 9. 슬라이싱

word = "Python"
print(word[1:4])

[1:4]는 인덱스 1부터 3까지 가져옵니다. 결과는 yth입니다.

코드 10. 단어 빈도 분석

words = ["AI", "Python", "AI", "Data"]
count = {}
for w in words:
    count[w] = count.get(w, 0) + 1
print(count)

get(w, 0)은 키가 없으면 0을 반환합니다. 단어가 나올 때마다 개수를 1씩 증가시킵니다.

🌟 수업마무리하기

자료구조는 데이터를 담는 그릇입니다. 문제 상황에 따라 적절한 그릇을 선택하면 코드가 훨씬 간단하고 강력해집니다.

  • 순서 있고 수정 가능한 데이터 → 리스트
  • 순서 있지만 수정 불가능한 데이터 → 튜플
  • 이름표가 붙은 데이터 → 딕셔너리
  • 중복 없는 데이터 → 집합
  • 문자의 순서 있는 모음 → 문자열

핵심 정리: 좋은 프로그래머는 데이터를 어떻게 저장할지 먼저 생각합니다.
