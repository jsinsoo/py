🚀 수업 들어가기

고급 파이썬 문법은 단순히 어려운 문법이 아니라, 짧고 읽기 쉽고 재사용 가능한 파이썬다운 코드를 작성하기 위한 도구입니다.

✨ 컴프리헨션

반복문과 조건문을 한 줄로 간결하게 표현합니다.

🔁 제너레이터

값을 한꺼번에 만들지 않고 필요할 때 하나씩 생성합니다.

🎁 데코레이터

기존 함수의 앞뒤에 기능을 덧붙입니다.

📦 언패킹

리스트, 튜플, 딕셔너리 값을 깔끔하게 풀어 사용할 수 있습니다.

영역목표
지식 목표컴프리헨션, generator, yield, iterator, decorator, closure, unpacking, *args, **kwargs의 개념을 설명할 수 있다.
기능 목표고급 문법을 활용하여 제너레이터, 로그 데코레이터, 가변 인수 함수, 코드 최적화 예제를 작성할 수 있다.
태도 목표파이썬다운 표현을 사용하되, 가독성과 명확성을 함께 고려하는 태도를 기른다.

🧠 핵심 개념

1. 컴프리헨션

컴프리헨션은 리스트나 딕셔너리를 짧고 직관적으로 생성하는 문법입니다.

2. generator와 yield

제너레이터는 값을 한 번에 모두 만들지 않고 필요할 때 하나씩 생성합니다. yield는 값을 잠시 반환하고 함수 실행 상태를 기억합니다.

3. iterator

이터레이터는 next()로 값을 하나씩 꺼낼 수 있는 객체입니다.

4. decorator

데코레이터는 함수를 감싸서 실행 전후에 새로운 기능을 추가합니다.

5. closure

클로저는 내부 함수가 외부 함수의 변수를 기억하는 구조입니다.

6. unpacking, *args, **kwargs

언패킹은 묶인 값을 풀어내는 문법이고, *args는 여러 위치 인수, **kwargs는 여러 키워드 인수를 받을 때 사용합니다.

💻 개념기반 학습코드 10개 + 상세 설명

코드 1. 리스트 컴프리헨션

nums = [x*x for x in range(1, 6)]
print(nums)

1부터 5까지의 제곱을 한 줄로 리스트에 저장합니다.

코드 2. 조건이 있는 리스트 컴프리헨션

even = [x for x in range(1, 11) if x % 2 == 0]
print(even)

1부터 10까지 중 짝수만 골라 리스트로 만듭니다.

코드 3. 딕셔너리 컴프리헨션

square = {x: x*x for x in range(1, 4)}
print(square)

숫자를 키로, 제곱을 값으로 갖는 딕셔너리를 만듭니다.

코드 4. generator와 yield

def gen():
    yield 1
    yield 2

g = gen()
print(next(g))
print(next(g))

yield는 값을 하나씩 생성합니다. next()를 호출할 때마다 다음 yield 값이 나옵니다.

코드 5. iterator

nums = iter([10, 20, 30])
print(next(nums))
print(next(nums))

iter()로 이터레이터를 만들고 next()로 값을 하나씩 꺼냅니다.

코드 6. decorator

def deco(func):
    def wrapper():
        print("시작")
        func()
        print("끝")
    return wrapper

@deco
def hello():
    print("Hello")

hello()

데코레이터는 hello 함수 실행 전후에 시작과 끝을 출력하도록 기능을 추가합니다.

코드 7. closure

def outer(x):
    def inner(y):
        return x + y
    return inner

add5 = outer(5)
print(add5(3))

inner 함수가 outer의 x=5를 기억합니다. 그래서 add5(3)은 8입니다.

코드 8. unpacking

a, b, c = [1, 2, 3]
print(a + b + c)

리스트의 값을 변수 a, b, c에 각각 풀어 저장합니다.

코드 9. *args

def total(*args):
    return sum(args)

print(total(1, 2, 3, 4))

*args는 개수가 정해지지 않은 여러 위치 인수를 튜플처럼 받습니다.

코드 10. **kwargs

def info(**kwargs):
    print(kwargs["name"], kwargs["score"])

info(name="Kim", score=95)

**kwargs는 여러 키워드 인수를 딕셔너리처럼 받습니다.

🧪 실습 예제

1. 제너레이터 만들기

def count_up(n):
    for i in range(1, n+1):
        yield i

for num in count_up(5):
    print(num)

2. 데코레이터 로그 시스템

def logger(func):
    def wrapper(*args, **kwargs):
        print("함수 실행:", func.__name__)
        result = func(*args, **kwargs)
        print("함수 종료:", func.__name__)
        return result
    return wrapper

@logger
def add(a, b):
    return a + b

print(add(3, 4))

3. 코드 최적화 예제

scores = [70, 85, 90, 40, 100]
passed = [s for s in scores if s >= 60]
average = sum(passed) / len(passed)
print(passed)
print(average)

🌟 수업마무리하기

고급 파이썬 문법은 코드를 무조건 짧게 쓰는 기술이 아니라, 의도를 명확하게 드러내고 반복을 줄이며 메모리를 효율적으로 사용하는 표현 방식입니다.

  • 컴프리헨션은 간결한 데이터 생성을 돕는다.
  • 제너레이터는 메모리를 절약하며 값을 하나씩 만든다.
  • 데코레이터는 기존 함수에 기능을 추가한다.
  • 클로저는 외부 변수의 값을 기억한다.
  • 언패킹은 묶인 데이터를 깔끔하게 풀어낸다.
  • *args와 **kwargs는 유연한 함수 설계를 가능하게 한다.

핵심 정리: 파이썬다운 코드는 짧기만 한 코드가 아니라, 읽기 쉽고 의도가 분명한 코드입니다.
