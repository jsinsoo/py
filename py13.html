01. 웹페이지 제목 가져오기

import requests
from bs4 import BeautifulSoup

url = "https://example.com"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

print("웹페이지 제목:", soup.title.text)

✍️ 출력 결과 서술형

example.com의 title 태그가 Example Domain이라고 가정할 때 출력 결과를 쓰시오.

정답
웹페이지 제목: Example Domain
상세 풀이

soup.title.text는 title 태그의 텍스트를 가져옵니다. print()는 앞 문자열과 title 텍스트를 공백으로 구분하여 출력합니다.

⭕❌ 개념 OX 퀴즈

BeautifulSoup(res.text, 'html.parser')는 HTML 문자열을 분석 가능한 객체로 바꾸는 역할을 한다.

정답: O
상세 풀이: BeautifulSoup은 HTML 문자열을 태그 단위로 탐색할 수 있는 객체로 변환합니다.

🧩 심화 5지선다형

웹페이지의 title 태그 텍스트를 가져오는 표현으로 가장 적절한 것은?

정답: 1번
상세 풀이: soup.title은 title 태그를 의미하고, .text는 태그 안의 텍스트를 가져옵니다.

02. 뉴스 제목 크롤링

import requests
from bs4 import BeautifulSoup

url = "https://news.ycombinator.com/"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

titles = soup.find_all("span", class_="titleline")

for i, title in enumerate(titles[:10], 1):
    print(i, title.text)

✍️ 출력 결과 서술형

titles에 제목 텍스트가 ['Alpha', 'Beta'] 두 개만 있다고 가정하고 titles[:10]을 반복하면 출력 결과를 쓰시오.

정답
1 Alpha
2 Beta
상세 풀이

enumerate(titles[:10], 1)은 번호를 1부터 붙입니다. 제목이 두 개뿐이면 두 줄만 출력됩니다.

⭕❌ 개념 OX 퀴즈

find_all('span', class_='titleline')은 조건에 맞는 span 태그들을 모두 찾는다.

정답: O
상세 풀이: find_all()은 조건에 맞는 모든 요소를 리스트처럼 반환합니다.

🧩 심화 5지선다형

enumerate(titles[:10], 1)에서 숫자 1의 의미는?

정답: 1번
상세 풀이: enumerate(iterable, start)는 번호의 시작값을 start로 지정합니다.

03. 모든 링크 주소 추출

import requests
from bs4 import BeautifulSoup

url = "https://example.com"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

links = soup.find_all("a")

for link in links:
    print(link.get("href"))

✍️ 출력 결과 서술형

HTML에 <a href='https://a.com'>A</a><a href='https://b.com'>B</a> 두 링크가 있다고 가정할 때 출력 결과를 쓰시오.

정답
https://a.com
https://b.com
상세 풀이

find_all('a')는 모든 a 태그를 찾고, get('href')는 각 링크의 href 속성값을 가져옵니다.

⭕❌ 개념 OX 퀴즈

a 태그의 href 속성에는 보통 이동할 링크 주소가 들어 있다.

정답: O
상세 풀이: HTML에서 a 태그는 하이퍼링크를 만들고 href 속성에 이동 주소를 넣습니다.

🧩 심화 5지선다형

태그의 href 속성을 안전하게 가져오는 코드로 가장 적절한 것은?

정답: 1번
상세 풀이: get('href')는 해당 속성이 없을 때도 None을 반환하여 비교적 안전합니다.

04. 쇼핑몰 상품 가격 수집

import requests
from bs4 import BeautifulSoup

url = "https://books.toscrape.com/"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

prices = soup.find_all("p", class_="price_color")

for price in prices[:10]:
    print(price.text)

✍️ 출력 결과 서술형

prices의 텍스트가 ['£51.77', '£53.74'] 두 개만 있다고 가정할 때 출력 결과를 쓰시오.

정답
£51.77
£53.74
상세 풀이

prices[:10]은 최대 10개를 의미하지만 실제 요소가 2개면 2개만 반복 출력됩니다.

⭕❌ 개념 OX 퀴즈

class_='price_color'에서 class_처럼 밑줄을 붙이는 이유는 class가 파이썬 예약어이기 때문이다.

정답: O
상세 풀이: class는 파이썬 문법에서 예약어이므로 BeautifulSoup에서는 class_ 매개변수를 사용합니다.

🧩 심화 5지선다형

p 태그 중 class가 price_color인 요소를 모두 찾는 코드는?

정답: 1번
상세 풀이: find_all(태그명, class_='클래스명') 형식으로 조건에 맞는 태그를 모두 찾습니다.

06. 크롤링 결과 CSV 저장

import requests
from bs4 import BeautifulSoup
import csv

url = "https://news.ycombinator.com/"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")
titles = soup.find_all("span", class_="titleline")

with open("news.csv", "w", newline="", encoding="utf-8-sig") as f:
    writer = csv.writer(f)
    writer.writerow(["번호", "제목"])

    for i, title in enumerate(titles[:10], 1):
        writer.writerow([i, title.text])

✍️ 출력 결과 서술형

titles에 'Alpha' 하나만 있다고 가정할 때 CSV에 데이터 행으로 저장되는 내용 형식을 쓰시오.

정답
1,Alpha
상세 풀이

writer.writerow([i, title.text])는 번호와 제목을 한 행에 저장합니다. CSV에서는 쉼표로 구분된 형태로 이해할 수 있습니다.

⭕❌ 개념 OX 퀴즈

utf-8-sig 인코딩은 엑셀에서 한글 CSV가 깨지는 문제를 줄이는 데 도움이 될 수 있다.

정답: O
상세 풀이: utf-8-sig는 BOM을 포함하여 일부 프로그램에서 한글 인식을 더 안정적으로 해 줍니다.

🧩 심화 5지선다형

CSV 파일에 한 줄을 기록하는 메서드는?

정답: 1번
상세 풀이: csv.writer 객체의 writerow()는 한 행을 CSV 파일에 기록합니다.

07. 날씨 정보 크롤링 구조

import requests
from bs4 import BeautifulSoup

url = "https://example.com/weather"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

temp = soup.find("span", class_="temp")

if temp:
    print("현재 기온:", temp.text)
else:
    print("기온 정보를 찾을 수 없습니다.")

✍️ 출력 결과 서술형

temp 요소를 찾지 못해 temp가 None이라고 가정할 때 출력 결과를 쓰시오.

정답
기온 정보를 찾을 수 없습니다.
상세 풀이

if temp 조건이 거짓이므로 else 블록이 실행되어 안내 문구가 출력됩니다.

⭕❌ 개념 OX 퀴즈

if temp:는 temp 요소를 찾았는지 확인하여 None일 때 오류를 방지하는 데 도움이 된다.

정답: O
상세 풀이: soup.find()가 실패하면 None을 반환하므로 조건문으로 확인하면 안전합니다.

🧩 심화 5지선다형

soup.find('span', class_='temp')가 요소를 찾지 못했을 때 일반적으로 반환되는 값은?

정답: 1번
상세 풀이: BeautifulSoup의 find()는 조건에 맞는 요소가 없으면 None을 반환합니다.

08. Selenium 로그인 자동화 구조

from selenium import webdriver
from selenium.webdriver.common.by import By
import time

driver = webdriver.Chrome()

driver.get("https://example.com/login")

driver.find_element(By.NAME, "username").send_keys("myid")
driver.find_element(By.NAME, "password").send_keys("mypassword")

time.sleep(2)

driver.quit()

✍️ 출력 결과 서술형

이 코드에서 username 입력창에 입력되는 문자열을 쓰시오.

정답
myid
상세 풀이

send_keys('myid')가 username이라는 name 속성을 가진 입력창에 myid를 입력합니다.

⭕❌ 개념 OX 퀴즈

driver.quit()은 브라우저를 종료하고 Selenium 세션을 정리한다.

정답: O
상세 풀이: driver.quit()은 자동화된 브라우저를 닫고 관련 자원을 정리합니다.

🧩 심화 5지선다형

Selenium에서 name 속성이 username인 요소를 찾는 코드로 가장 적절한 것은?

정답: 1번
상세 풀이: Selenium은 find_element(By.NAME, 값) 형식으로 name 속성 기준 요소를 찾을 수 있습니다.

09. 무한 스크롤 페이지 크롤링

from selenium import webdriver
import time

driver = webdriver.Chrome()
driver.get("https://example.com")

for _ in range(5):
    driver.execute_script(
        "window.scrollTo(0, document.body.scrollHeight);"
    )
    time.sleep(2)

print(driver.page_source)

driver.quit()

✍️ 출력 결과 서술형

for _ in range(5):에 의해 스크롤 명령은 몇 번 실행되는지 쓰시오.

정답
5
상세 풀이

range(5)는 0부터 4까지 총 5번 반복하므로 스크롤 명령도 5번 실행됩니다.

⭕❌ 개념 OX 퀴즈

execute_script()는 브라우저 안에서 JavaScript 코드를 실행할 수 있다.

정답: O
상세 풀이: Selenium의 execute_script()는 현재 페이지에서 JavaScript를 실행합니다.

🧩 심화 5지선다형

페이지 맨 아래까지 스크롤하는 JavaScript 명령으로 가장 적절한 것은?

정답: 1번
상세 풀이: window.scrollTo(0, document.body.scrollHeight)는 문서의 아래쪽 끝까지 스크롤합니다.

10. 크롤링 → Pandas 분석

import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "https://books.toscrape.com/"
res = requests.get(url)

soup = BeautifulSoup(res.text, "html.parser")

titles = soup.find_all("h3")

data = []

for t in titles:
    data.append(t.text.strip())

df = pd.DataFrame(data, columns=["책 제목"])

print(df)

✍️ 출력 결과 서술형

data = ['Book A', 'Book B']일 때 DataFrame의 열 이름은 무엇인지 쓰시오.

정답
책 제목
상세 풀이

pd.DataFrame(data, columns=['책 제목'])에서 columns 인자로 지정한 열 이름은 책 제목입니다.

⭕❌ 개념 OX 퀴즈

Pandas DataFrame은 표 형태 데이터를 다루는 데 사용할 수 있다.

정답: O
상세 풀이: DataFrame은 행과 열로 구성된 표 형태 데이터 구조입니다.

🧩 심화 5지선다형

리스트 data를 '책 제목' 열을 가진 DataFrame으로 만드는 코드는?

정답: 1번
상세 풀이: pd.DataFrame()에 data와 columns를 전달해 표 형태 데이터로 변환합니다.
